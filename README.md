# Cities wishlist

Web app for travel planning

## Setup
### Create .env file containing:
```sh
CONTAINER_NAME=
POSTGRES_USER=
POSTGRES_PASSWORD=
POSTGRES_DBNAME=
DB_PORT=
NGINX_PORT=
SECRET_KEY=
```
### Run Docker containers
```sh
docker compose up 
```
if it's the first time, run
```sh
docker compose up --build
```
